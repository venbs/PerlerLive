# PerlerStudio 拼豆工作室

## 1. 概述
拼豆工作室是一个基于P5.js的拼豆设计与创作工具。用户可以导入图片，将图片转换为拼豆图，并进行编辑和保存。

## 2. 原理流程

> 以下程序原理仅为初步构想，如果在开发过程中有更好的想法，可以随时调整。

### 2.1 缩放采样
用户导入一张图片，程序将图片向下采样到用户设置的分辨率，保留对应的像素信息。

### 2.2 颜色量化
从原始图片的颜色中提取颜色，并根据用户选择的颜色数量，对颜色进行聚类，用户可以选择用几色拼豆，有4色、8色、16色、32色可选，并以此为基础创建色板。

### 2.3 拼豆填充
将量化后的色板中的颜色，映射到拼豆上，并且拼豆在填充时，可以选择抖动，从而创造更平滑的过渡和保留更丰富的细节。

### 2.4 拼豆样式
拼豆圆角：用户可以拖动滑块调整拼豆的圆角尺寸，圆角尺寸为0-100%，默认值为50%，数值为0时，拼豆为正方形，数值为100时，拼豆为圆形。

拼豆豆心：豆心效果是在拼豆内部添加一个半透明圆点，圆点的大小为拼豆大小的0%-50%，默认值为20%，圆点的位置为拼豆的中心，豆心的默认颜色为20%不透明度的黑色。

拼豆间隔：用户可以拖动滑块调整拼豆的间隔大小，间隔大小为0.5-3px，默认值为1px，数值为0.5时，拼豆间隔为0.5px，数值为3时，拼豆间隔为3px。



## 3. UI设计

### 3.1 布局
整体界面非常简洁，左上角是"PerlerStudio"的logo。
画面中间空白区域是画布，画布是平整的，不会有明确的边框，画布是纯白色的，上面有淡淡的点阵，画布可以拖拽也可以缩放。
右侧是一个悬浮面板，它有以下元素：
1. 导入图片按钮
2. 拼豆分辨率选择，单边最小分辨率为16，最大为256（默认值为64）。
3. 拼豆的颜色数量选择，有4色、8色、16色、32色可选（默认值为16）。
4. 拼豆色板，会列出当前色板中的所有颜色，用户可以点击颜色进行编辑调整当前的颜色，
5. 抖动开关，打开后，会对颜色进行抖动过渡（默认值为关闭）。
6. 拼豆像素的样式设置，包含以下参数：
    - 拼豆的圆角大小（0-100%），默认值为50%
    - 拼豆豆心的大小（0%-50%），默认值为20%
    - 拼豆之间的间隔间距（0.5-3px），默认值为1px
7. 导出图片按钮，点击按钮后，会出现下拉按钮，分别是导出PNG，导出SVG。
悬浮面板不可以响应拖拽。

### 3.2 界面风格
画面风格极简，风格为极简light（浅色），大圆角按钮。
整体界面精致，UI高级。

## 4. 技术栈
- P5.js （可能需要包含p5.svg库）
- HTML
- CSS
- JavaScript

为WebApp，暂不考虑为移动端适配，尽量保证在1920*1080分辨率下有较好的体验。
