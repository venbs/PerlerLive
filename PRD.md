# PerlerStudio 拼豆工作室

## 1. 概述
拼豆工作室是一个基于P5.js的拼豆设计与创作工具。用户可以导入图片，将图片转换为拼豆图，并进行编辑和保存。

## 2. 原理流程

> 以下程序原理仅为初步构想，如果在开发过程中有更好的想法，可以随时调整。

### 2.1 图像预处理
为了达到最好的拼豆效果，需要对用户导入的图片进行预处理，预处理包括以下步骤：
1. 为了防止输入的图片本身存在纯黑像素，导致后续厚度效果时黑色像素的细节完全不可见，需要将图片的黑色像素进行微调，可以将整体暗部像素的亮度进行极其细微的提升，这个行为默认进行后台处理，用户无法关闭。
2. 为了后续制作拼豆时有更好的画面效果，对输入图像使用卡通滤镜处理(p5.FIP/cartoon滤镜)，这样会生成一个描边，更适合进行拼豆的效果，这个效果由开关控制。


### 2.2 缩放采样
用户导入一张图片，程序将图片向下采样到用户设置的分辨率，保留对应的像素信息。

### 2.3 颜色量化
从原始图片的颜色中提取颜色，并根据用户选择的颜色数量，对颜色进行聚类，用户可以选择用几色拼豆，有4色、8色、16色、32色与自定义可选，并以此为基础创建色板。

颜色色板允许用户点击颜色进行调整，单击色块，会呼出颜色选择器，用户可以选择新的颜色，新颜色提交后，会对比现有色板，如果与现有色板中的某个颜色一致，则会合并这两个颜色，否则，将新颜色添加到色板中，并更新。

当调整颜色后导致颜色合并，颜色总数量小于用户选择的数量时，则颜色数量选择器会切换到“自定义”模式。
如果用户再次点击4、8、16、32中的任意一个，则会重新进行颜色量化，并更新色板。

### 2.4 拼豆填充
将量化后的色板中的颜色，映射到拼豆上，并且拼豆在填充时，可以选择抖动，从而创造更平滑的过渡和保留更丰富的细节。

### 2.5 拼豆样式
拼豆圆角：用户可以拖动滑块调整拼豆的圆角尺寸，圆角尺寸为0-100%，默认值为50%，数值为0时，拼豆为正方形，数值为100时，拼豆为圆形。

拼豆厚度（浮雕）：用户可以拖动滑块调整拼豆边缘的浮雕厚度，厚度范围为0-20%，默认值为10%。浮雕效果通过对豆子边缘绘制斜向线性渐变描边实现（左上角为受光面提亮，右下角为暗部压暗），同时豆心的描边方向反转，产生内凹浮雕效果。设置为0时，关闭浮雕效果。

拼豆豆心：豆心效果是在拼豆内部添加一个半透明圆点，圆点的大小为拼豆大小的0%-50%，默认值为20%，圆点的位置为拼豆的中心，豆心的默认颜色为20%不透明度的黑色。

拼豆间隔：用户可以拖动滑块调整拼豆的间隔大小，间隔大小为0.5-3px，默认值为1px，数值为0.5时，拼豆间隔为0.5px，数值为3时，拼豆间隔为3px。


## 3. UI设计

### 3.1 布局
整体界面非常简洁，左上角是"PerlerStudio"的logo。
画面中间空白区域是画布，画布是平整的，不会有明确的边框，画布是纯白色的，上面有淡淡的网格线，网格线颜色为#e0e0e0，网格需要与豆子对齐。
顶部和左侧贴边有一个标尺，标尺单位为像素，1像素对应1个豆子，标尺的起点为左上角第一颗豆子为0，往右为正，往下为正，当前标尺范围内存在豆子时，标尺的颜色会略有区别，类似与figma的标尺，会高亮出来。拖动画布时，标尺也会随之移动。
右侧是一个悬浮面板，它有以下元素：
1. 导入图片按钮
2. 拼豆分辨率输入框，单边最小分辨率为16，最大值为128，（默认值为64）。
3. 描边开关，打开后，会对输入图像进行描边（p5.FIP/cartoon）处理（默认值为关闭）。开启后会出现“描边粗细”参数（0.001-0.015），用于实时调整轮廓线条宽度。
4. 拼豆的颜色数量选择，有4色、8色、16色、32色可选（默认值为16）。
5. 拼豆色板，会列出当前色板中的所有颜色，用户可以点击颜色进行编辑调整当前的颜色。
    - 色板的布局方式为list，每条list包含一个颜色块，颜色块的右侧是颜色的十六进制值，最后会显示当前颜色豆子一共使用了多少颗。
    - 可以直接点击色块呼出pickr进行色彩编辑，也可以直接调整后面的十六进制进行编辑。
    - 色板区域是一个scroll区域，当颜色数量超过面板高度时，会出现滚动条。
6. 抖动开关，打开后，会对颜色进行抖动过渡（默认值为关闭）。
7. 拼豆像素的样式设置，包含以下参数：
    - 拼豆的圆角大小（0-100%），默认值为50%
    - 拼豆浮雕厚度（0-20%），默认值为5%
    - 拼豆豆心的大小（0%-60%），默认值为40%
    - 拼豆之间的间隔间距（0.5-3px），默认值为1px
8. 导出图片按钮，点击按钮后，这是两个按钮组合的区域，分别是导出PNG，复制SVG。
    -  点击复制svg后，屏幕中心下方会有一个黑色半透明toast提示"SVG已复制到剪贴板"，持续时间为2秒。
悬浮面板不可以响应拖拽。

### 3.2 界面风格
画面风格极简，风格为极简light（浅色），大圆角按钮。
整体界面精致，UI高级。

## 4. 技术栈
- P5.js （可能需要包含p5.svg库）
- HTML
- CSS
- JavaScript

为WebApp，暂不考虑为移动端适配，尽量保证在1920*1080分辨率下有较好的体验。

## 5. 补充
- 考虑支持PWA，方便用户离线使用。
- 允许用户从剪贴板粘贴图片到画布上，从而快速完成导入操作，但如果粘贴的是非图片内容，则不做任何响应。
